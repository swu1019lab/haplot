# haplot

A set of easy-to-use plot function can be used for haplotype visual analysis

## Functions
- [x] manhattan and qq plot
- [x] LD heatmap plot
- [x] haplotype geographic distribution plot
- [x] gene structure plot
- [x] haplotype boxplot
- [x] neucleotide diversity and Fst plot

## Dependencies
- python >= 3.10
- numpy >= 1.22.3
- pandas >= 1.4.2
- matplotlib >= 3.6.2
- scipy >= 1.9.3
- geopandas >= 0.13.2
